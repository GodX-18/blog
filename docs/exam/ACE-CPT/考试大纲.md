# 考试内容大纲

## 目的

美国运动医学会(ACE)个人训练师认证考试的目的在于评估个人是否具备安全且称职地作为ACE认证个人训练师进行实践所必需的知识和技能。为确保ACE个人训练师认证保持相关性和时效性，ACE每五年进行一次角色界定研究（RDS），以更新ACE个人训练师认证项目和考试的内容构成和重点。

2022年7月，ACE与Scantron Corporation（现为Meazure Learning），一家提供认证、许可设计、开发和测试服务的公司，对ACE个人训练师认证进行了RDS研究，以确定个人训练师在日常工作中执行的主要任务。

这次RDS的基本目的是建立和验证ACE个人训练师认证考试适当的内容领域。这个过程的结果包括这份考试内容大纲，它为每次ACE个人训练师认证考试提供了一个蓝图。

在准备考试时，重要的是要记住，所有考试问题都是基于这个大纲的。 这份蓝图包括个人训练师在职业入门级胜任力水平上执行工作职责所需的任务、知识和技能。这些任务、知识和技能是由个人训练主题专家RDS小组确定的，然后发送给数千名ACE认证个人训练师进行验证。 

ACE的立场是，这里概述的建议并不详尽地涵盖个人训练师的资格，但代表了一个合格且安全地作为个人训练师实践所需的最小熟练度和理论知识水平。所有任务都会在每次考试中进行评估；然而，重要的是要注意，考试内容大纲中列出的所有知识和技能陈述并不都会在每次考试中涉及。

:::warning 考试考生请注意

在准备ACE认证考试时，请注意，由于健身行业的不断发展和当前及正在进行的研究的新进展，任何教材中的内容都可能过时。这些考试基于深入的职业分析和行业范围内的验证调查。按照设计，这些考试评估的是考生对最新科学依据的专业标准和指南的知识和应用。这一领域的动态性质要求ACE认证考试定期更新，以确保它们反映最新的行业发现和研究。因此，通过这些考试所需的知识和技能并不仅仅体现在任何单一教材中。除了学习ACE教材中的内容外，ACE强烈建议所有考试候选人和健身专业人士关注来自各种有效行业来源的新进展、指南和标准。

:::

## 描述

2022年个人训练师角色界定研究定义了当前考试内容大纲中的任务、知识和技能陈述。

* 任务是指完成工作或职业的某些方面所需的个体功能，无论是心理的还是身体的。
* 领域是工作或职业的主要责任或活动区域，它们代表了任务的逻辑分组。
* 知识和技能陈述列出了执行特定任务所需的知识和/或技能。它们包括候选人必须具备的信息、行动或其他可学习的技能，以便执行特定任务。事实或程序信息的有序集合称为知识，而熟练的身体、口头或心理操作数据、人或物体称为技能。 

ACE个人训练师认证考试的考试内容大纲包括14个任务，分为四个领域。通过RDS过程，每个领域都被分配了一个权重，这代表了在每个考试中该领域内将出现的考试问题的近似百分比。以下列出了ACE个人训练师认证考试的考试内容大纲，包括每个领域及其相关百分比权重、每个领域内的任务，以及与执行每个任务相关的知识和技能陈述。

## 领域 I：客户入门和评估（23%）

### 任务 1

通过问卷调查、访谈和其他适当文件获取健康、医疗、运动和生活信息，以确定运动参与的准备和风 险，识别医疗许可和转诊的需求，并促进项目设计。

**知识：**

1. 有效沟通策略（例如，积极倾听、语言和非语言） 
2. 年龄、文化、能力（例如，身体、认知）和其他因素（例如，个人、社会、身体活动属性） 对建立关系和相关健康和生活信息收集的影响
3. 用于收集项目设计和实施所需信息的工具（例如，运动前健康筛查、动机访谈、豁免声明） 和方法（例如，电话、虚拟、面对面）
4. 适当膳食评估方法及其目的、固有风险和益处 
5. 膳食偏好（例如，纯素食、停止高血压饮食方法、低碳水化合物）及其对锻炼的潜在影响
6. 人群特定（例如，老年人、孕产期、青年）和疾病特定（例如，腰背问题、糖尿病、癌症）的健康和 锻炼注意事项及禁忌症 
7. 心血管和其他慢性疾病或条件的风险因素 
8. 常用药物对锻炼的生理反应（例如，心率、血压）的影响 

**技能：**

1. 与多样化人群（例如，年龄、文化、身体和认知能力）建立融洽关系、建立信誉和建立信任
2. 在运动参与之前进行有效访谈并收集相关健康和生活信息。

3. 根据行业标准和使用指南（例如，PAR‑Q+、美国运动医学会指南）解释访谈和问卷响应，以 确定运动参与的禁忌症/特殊注意事项，从而促进项目设计。 
4. 识别需要并适当获取医疗许可和转诊

### 任务 2

评估客户进行健康相关行为改变的准备情况，并通过有效沟通评估锻炼态度和信念，以建立融洽关系并设定适当目标

**知识：**

1. 有效沟通策略（例如，积极倾听、语言和非语言） 
2. 工具（例如，运动史和态度问卷、改变准备性问卷）和方法（例如，动机访谈）用于确定客户 态度、偏好和行为改变准备度 
3. 参与运动和改变行为的常见障碍和动机因素 
4. 行为改变理论（例如，阶段改变理论、健康信念模型） 
5. 目标类型和目标设定过程（例如，SMART目标、GROW模型）

**技能：**

1. 建立融洽关系、建立信誉和建立信任 
2. 应用有效的动机访谈策略（例如，开放式问题、澄清、探究） 
3. 确定客户态度、偏好和行为改变的准备度，以及激励项目采纳和依从性的适当策略 
4. 与客户合作，有效设定目标（例如，SMART目标、GROW模型）

### 任务3

通过观察和评估识别并评估基础运动的质量，以制定适当的锻炼计划，从而提高功能性运动、健康、 体能和表现。

**知识：**

1. 运动相关解剖学和运动学（例如，主要肌群、关节动作） 
2. 运动物理定律（例如，力量产生、动量） 
3. 生物力学、关节活动度与稳定性及其对姿势和运动的影响 
4. 基础运动模式（例如，弯举和举重、单腿、推、拉、旋转） 
5. 评估关节位置、稳定性和活动度的有效方法，以确定其对姿势、运动和平衡的影响

**技能：**

1. 识别日常活动对运动效率的影响（例如，生活方式、职业性、运动性） 
2. 观察和识别夸张的姿势偏差 
3. 选择和实施评估关节活动度和稳定性的适当方法 
4. 选择和实施评估运动和平衡的适当方法 
5. 解释姿势观察、运动筛查和其他评估的结果，以促进项目设计

### 任务4

根据客户访谈、问卷调查和标准化协议选择和进行基线体能评估，以促进安全有效的项目设计并监控随 时间的变化。

**知识：**

1. 运动测试协议、方法以及在不同环境中（例如：虚拟、面对面、居家、俱乐部、公共空间）使用适当 技术的应用 
2. 体能评估（例如：心肺功能、肌肉力量和耐力、柔韧性、身体成分）的适当应用和顺序 
3. 急性运动参与时的生理反应以及负荷变化 
4. 影响运动急性生理反应的变量（例如，年龄、性别、健康状况、环境条件、水合作用、药物） 
5. 有效沟通策略（例如，提示、语言和非语言）

**技能：**

1. 根据个体客户需求和环境，选择、实施和修改适当的体能评估（例如，心肺功能、肌肉力量和耐力、 柔韧性、身体成分） 
2. 识别急性生理反应并应用测试终止标准 
3. 评估和解释体能评估结果以促进项目设计 
4. 传达体能评估结果 
5. 利用适当且可用的技术（例如，心率监测器、健康应用程序）

## 领域 II：项目设计与实施（31%）

### 任务1

通过解释客户访谈和评估数据，建立适当的功能、健康、体能和/或表现目标，以设计个性化的运动计划。

**知识：**

1. 运动相关解剖学、生理学和生物力学（例如，主要肌群、关节动作、心肺反应） 
2. 营养与体重管理原则和指南（例如，能量、营养素、时机、水合作用） 
3. 解读客户健康史、风险因素、运动准备度及其他评估数据的指南 
4. 用于促进行为改变的有效目标设定策略（例如，SMART目标、GROW模型） 
5. 影响运动参与和依从性的个人因素（例如，个人特质、身体活动、动机、过往经验） 
6. 健康的社会决定因素（例如，经济稳定性、教育水平、社会和社区环境、医疗保健的可及性、健康素养、社区和健身环境

**技能：**

1. 解释对评估和锻炼的响应（例如，生理的、心理的、情绪的） 
2. 将对访谈和评估的响应整合到锻炼计划设计中 
3. 结合来自各种评估的发现，使用循证建议（例如，心肺功能、柔韧性、肌肉力量和耐力、身体成分、血压）
4. 有效传达评估结果 
5. 促进有效目标设定（例如，SMART目标、过程、结果、GROW模型） 
6. 基于访谈、评估、生活方式和目标设计个性化运动计划

### 任务2

通过应用适当的运动原则和指南设计个性化运动计划，以增强功能、健康、体能和表现

**知识：**

1. 运动相关解剖学、生理学和生物力学（例如，主要肌群、关节动作和活动范围、心肺反应、运动平面） 
2. 体能成分（例如，心肺功能、身体成分、肌肉力量和耐力、柔韧性） 
3. 一般运动和计划设计原则（例如，超负荷、特异性、边际效益递减、恢复、可逆性、进展） 
4. 监测个体对心肺运动的个性化反应的适当方法（例如，心率、通气阈值、自我感觉劳累程度） 
5. 运动引起的急性和慢性生理和心理适应 
6. 行业指南关于运动计划变量（例如，频率、强度、时间、类型、量、模式、进展） 
7. 适宜的运动方案和技术（例如，热身、放松、运动准备、恢复、肌筋膜放松） 
8. 制定和发布锻炼循证标准和指南的专业组织和政府机构（例如，ACSM、ACOG、美国糖尿病协会、 美国心脏协会、NIH、美国疾病控制与预防中心、美国儿科学会） 
9. 与技能相关的体能的组成部分（例如，平衡、敏捷性、速度、力量、反应性、协调性）

**技能：**

1. 根据目标设计个性化运动计划（例如，心肺功能、肌肉力量和耐力、表现、活动度、稳定性、平 衡、柔韧性、身体成分、整体健康） 
2. 根据个性化功能、健康、体能和表现选择和排列适宜的练习 
3. 根据个性化锻炼反应（例如：心率、通气阈值）设计适宜的心肺功能训练计划 
4. 实施适宜的方案组成部分（例如：热身、心肺功能、肌肉力量和耐力、放松、恢复）和有效的锻炼技术 
5. 根据需要修改练习技术，以促进安全锻炼参与和目标达成 
6. 修改指导及程序组件，以适应不同环境（例如：虚拟、面对面、居家、俱乐部、公共空间）中健康和/或体能状态个体差异

### 任务3

根据循证研究和指南，选择合适的锻炼和设备，并将其整合到客户项目中，以改善功能、健康、 体能、表现和整体健康

**知识：**

1. 一般锻炼和项目设计原则（例如，超负荷、特异性、边际效益递减、恢复、可逆性、进展） 
2. 为实现任务和目标特定结果所需的适当锻炼和设备选择  
3. 监测个体对心肺运动个性化反应的适当方法（例如，心率、通气阈值、自觉运动强度） 
4. 锻炼的急性与慢性生理和心理适应 
5. 运动物理定律（例如：惯性、加速度、动量、冲击力和反作用力、杠杆分类、力量产生） 
6. 最佳关节稳定性和活动度、运动模式及其与姿势和动力链的关系（例如：平衡、关节对齐、显著的肌肉 不平衡） 
7. 进展或退步运动强度或复杂性的策略和技巧（例如：支撑基础的改变、负荷量、拉力线、速度、重心）

**技能：**

1. 识别运动模式偏差 
2. 识别日常生活活动和生活方式（例如，职业、运动）的生理和心理需求 
3. 通过观察和反馈，在评估和锻炼过程中识别生理和心理反应 
4. 修改指导程序组件，以适应健康和/或体能状况 
5. 根据个性化锻炼反应（例如，心率、通气阈值）设计适宜的心肺功能训练计划 
6. 将适当的锻炼和设备结合到程序设计中 
7. 实施安全有效的运动计划，以解决肌肉不平衡并增强姿势对齐

### 任务4

通过语言、视觉和本体感觉提示指导客户安全有效的运动技巧和设备使用，以促进依从性并实现计 划目标。

**知识：**

1. 在不同环境中（例如，虚拟、面对面、居家、俱乐部、公共空间）使用各种设备、技术和方式方法进行适当的运动技巧 
2. 沟通策略（例如，积极倾听、语言和非语言沟通） 
3. 提示形式（例如，运动示范、口头指导、触觉纠正） 
4. 用于促进客户安全和计划成功的教育策略 
5. 测量与监测锻炼强度的方法 
6. 适当的呼吸技巧

**技能：**

1. 利用与个人客户需求和偏好相一致的沟通策略（例如，学习风格、感官需求、知识、态度、信 念、文化） 
2. 根据需要修改练习技术以促进目标达成 
3. 识别和纠正代偿性运动模式 
4. 在不同环境中（例如，虚拟、面对面、居家、俱乐部、公共空间）使用各种设备、方式方法和反 馈教授安全有效的运动技巧 
5. 利用现有技术在项目实施中（例如，数字流媒体、应用程序、软件）

## 领域 III：程序修改与进展 27%

### 任务1

通过为客户提供积极的体验，增强他们的自我效能感，以动机、教育和项目调整来实现以客户为中心的目标，从而促进项目依从性。 所需知识：

**知识：**

1. 当前的健身方法、产品和服务（例如，小组训练、设备、服装）及其相关的风险和益处 
2. 影响行为改变计划依从性的因素（例如，个人特质、身体活动、环境、社会经济、自我效能） 
3. 不同的环境中促进客户目标依从性的沟通方式和技巧（例如，虚拟、面对面、居家、俱乐部、 公共空间） 
4. 对锻炼的急性、慢性生理和心理适应 
5. 进展或退步锻炼动作强度或复杂性的策略和技巧（例如，支撑基础的改变、负荷量、拉力线、速度、 重心）、心肺运动和总工作量 
6. 行为改变模型和人类发展工具/理论（例如，阶段改变理论、DISC模型、决策平衡） 7
7. 支持健康生活方式改变的可靠资源（例如，循证在线锻炼图书馆、营养信息） 
8. 增强安全性、归属感和包容性的因素 
9. 常见饮食和补充剂的利弊

**技能：**

1. 维持融洽关系、可信度和信任 
2. 教育客户关于基础营养和体重管理指南 
3. 教育客户如何管理各种锻炼的频率、强度和复杂性 
4. 教育客户如何管理影响计划依从性的因素（例如，个人特质、身体活动、环境、社会经济、自 我效能） 
5. 根据需要修改运动技术、选择和强度，以确保安全并促进目标达成 
6. 激励客户坚持运动计划 
7. 创建安全且包容的环境以提高自我效能和归属感
8. 通过使用多种行为改变或教练策略（例如，计划调整、强调客户优势、积极倾听、反馈）帮 助客户建立自信、自我效能和自主权 
9. 帮助客户了解自身身体状况（例如，姿势、运动姿势、运动）以促进自我意识 
10. 教育客户关于常见饮食和补充剂的利弊风险

### 任务2

通过承认障碍和促进因素，识别并应对计划依从性中的失误，并协同识别解决方案以促进客户持续参与。

**知识：**

1. 影响项目参与和依从性的因素（例如，心理的、生理的、社会的、环境的） 
2. 影响沟通、生活方式和偏好的个人因素或特征 
3. 减轻失误和克服锻炼项目依从性常见障碍的策略

**技能：**

1. 识别影响项目依从性和目标达成的潜在障碍和促进因素 
2. 实施策略以帮助客户克服障碍 
3. 调整项目变量（例如，组数、次数、强度、休息时间、节奏）以促进依从性 
4. 使用合适的语言和非语言沟通技巧 
5. 识别需要转介给合适的相关健康专业人士的需求

### 任务3

通过观察、客户反馈和数据常规评估项目有效性，以协作修改项目。

**知识：**

1. 营养评估和协议（例如，膳食回忆、食物记录、问卷调查），这些均在私人教练的执业 范围内 
2. 适当的体能评估 
3. 根据观察、反馈和数据，对程序组件进行适当的修改 
4. 心肺功能、肌肉力量和耐力以及柔韧性项目的行业指南，适用于适当的进展和退步（例如，频率、强度、 时间、类型、量、模式） 
5. 与技能相关的体能的行业指南，适用于适当的进展和退步（例如，平衡、敏捷性、速度、力 量、反应性、协调性） 
6. 主要肌群、运动平面、关节动作和活动范围 
7. 主要运动模式（例如，弯举和举重、单腿、推、拉、旋转）以及适当的变体和修改 
8. 适合最佳身体力学的锻炼技术 
9. 适合预期结果的锻炼和强度（例如，负荷、自我感觉劳累程度、通气阈值） 
10. 运动引起的生理和心理反应
11. 执行各种类型运动（如力量训练、普拉提、瑜伽、太极拳、气功）的适当呼吸技巧 
12. 终止运动的条件

**技能：**

1. 选择和管理适当的评估（例如，运动、姿势、体能、营养、行为）以评估进展并根据需要调整计划目标 
2. 收集和解释客户数据和反馈以评估进展并根据需要调整计划目标 
3. 识别过度训练的迹象和症状（例如，身体表现下降、食欲改变、过度疲劳、情绪障碍） 
4. 根据客户功能、健康、体能和表现，选择锻炼、设备和负荷，以便根据需要推进和退回运动计划 
5. 通过沟通（例如，开放式问题、肯定、反思、总结、动机访谈）激发客户的运动准备度 
6. 通过观察（例如，肢体语言、运动模式、情绪监测、积极倾听）识别客户的运动准备度 
7. 识别对锻炼的生理和心理反应的变化（例如，表现、对施加的需求的反应、情绪、运动模式偏差） 
8. 根据观察和客户反馈，在不同的环境（例如，虚拟、面对面、居家、俱乐部、公共空间）中修改锻炼、技 巧和程序变量 
9. 识别终止锻炼的迹象

## 领域 IIII：风险管理、专业行为和道德商业实践（19%）

### 任务1

在各种环境中（例如，虚拟、面对面、居家、俱乐部、公共空间）根据公认的标准、指南、法律和法 规应用风险管理策略，以最大限度地减少责任并保护客户、私人教练和其他相关方。

**知识：**

1. 适用的指南、标准、法律和法规（例如，美国疾病控制与预防中心、美国运动医学会、职业安全与健康管 理局、公平劳动标准法、健康保险流通与责任法案、家庭教育权利和隐私法案、国家心理健康福祉委员会） 
2. 在执业范围内紧急响应的基本程序（例如，心肺复苏、自动体外除颤器、急救、应急行动计划、危 机管理） 
3. 提供运动计划的最佳实践 
4. 预防伤害和安全的最佳实践（例如，视线、布置、设备状况、保护、运动环境、紧急出口） 
5. 提供运动计划的行业指南（在各种环境条件下，例如，海拔、极端天气、温度、湿度、空气污染） 
6. 导致疏忽的因素（例如，作为行为、不作为行为） 
7. 风险管理规程（例如，识别、评估、方法、实施、监控） 
8. 通过安全数据存储和沟通渠道（例如，事件报告、事故报告、豁免声明、知情同意、SOAP记录）维护保 密性和隐私的方法

**技能：**

1. 根据推荐的行业或设施协议识别和报告潜在危险 
2. 进行风险评估（例如，识别、评估、方法、实施、监控）以确保安全锻炼参与 
3. 指导和监督安全有效的锻炼以及运动器材的使用 
4. 在锻炼过程中使用安全辅助技术帮助客户 
5. 转介客户至合适的医疗专业人员 
6. 识别并实施适当协议以确保技术的安全使用（例如：沟通、营销、数据跟踪、同意） 
7. 识别、应对并记录紧急情况及安全风险 
8. 根据极端环境条件（例如：海拔、极端天气、温度、湿度、空气污染）修改锻炼课程 
9. 选择合适的风险管理方法（例如：避免、转移、减少、保留）

### 任务2

保持更新以适应不断发展的健康和健身行业标准及与工作相关的执业范围，以增强专业信 誉和信任。

**知识：**

1. 执业范围、ACE伦理准则和专业行为 
2. 制定和发布循证研究和指南的专业组织和政府机构（例如，ACOG、ACSM、DHHS、USDA、 NIH、CDC、AND） 
3. 可信的继续教育提供者、项目及资源（例如，认证机构、会议、线上和线下课程、科学和专业期刊） 
4. 关于沟通平台、隐私保护和技术的正确使用的专业行为（例如，社交媒体、电子邮件、短信、应用程序、 可穿戴设备） 
5. 公平、多样性、包容性和文化能力

**技能：**

1. 区分可信与不可信资源，以保持对循证研究、理论和实践的更新 
2. 基于个人需求传播相关循证运动方案和教育信息（例如，社交媒体、博客、社区团体） 
3. 根据客户与私人教练协议选择合适的沟通渠道 
4. 展示促进公平、多样性、包容性和文化能力的适当行为 
5. 与客户和同事确定适当关系和确定专业界限

### 任务3

在各种健康和健身环境中（例如，虚拟、面对面、居家、俱乐部、公共空间）持续学习不断发 展的商业道德实践，以提升客户体验。

3. 商业计划组成部分（例如，执行摘要、产品和服务、市场分析、战略和实施） 
4. 营销计划的基本原则（例如，商业举措、目标市场、策略、渠道）
