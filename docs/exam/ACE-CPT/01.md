# 第一章：私人教练的角色和执业范围

:::tip 学习目标

* 定义与定期身体活动相关的健康益处
* 解释ACE认证的私人教练的角色和执业范围
* 描述在健康照护联盟中私人教练与其他专业人员之间的异同
* 确定何时何地将客户转介给其他健康照护联盟的专业人员
* 列出私人教练获得专业认可和职业发展的关键因素

:::

* 2018年美国运动指南：上一个版本2008年，十年更新一次，基本相似。
* 第一条：“500—1000MET—分钟”，发现人是梅脱。发现当一个人在安静状态下，身体当中需要3.5ml/kg/min的氧气，3.5ml/kg/min=1MET。
  * 中等强度=3met、剧烈强度=6met+.500-1000MET=150-300mins
* 第二条：150—300分钟，对应500—1000MET。对会员宣传，选择通俗易懂的150—300分钟中高强度的有氧运动。除了有氧，也需要做每周2次的肌肉强度锻炼，以保持健康。
  * 心肺相关+力量相关=真正的健康
* 第三条：6—17岁，需要进行60分钟的中高强度的运动/每天。
  很多小孩达不到这个水平，成为运动缺乏，需要有运动的经历，促进孩子的健康发展。
  * 孩子养成体育运动经历，长大更容易养成习惯。
* 第四条：降低血压和胆固醇水平，建议每周3—4次中高强度有氧运动，40分钟/次。
  * 对血压和胆固醇水平的控制有这良好的帮助。
  * 特殊人群运动之前需要经过医生同意，指南是一般建议，需要考虑个体差异。
* 美国运动指南的运动形式示例：
  * 步行，园艺（偏美国本土，只是一种形式），徒步旅行（这部分的客户进入健身房相对较少，但是教练可以参与体验，增进同理性），舞蹈，自行车（共享单车），体育游戏（如果孩子在课间会参与一些球类运动，可以满足每日60分钟），游泳（3—6MET属于中等强度，泡在水里也会达到6MET，环境不同的差异，对于关节问题的客户比较适宜）
  * 警示：过早死亡中有10%可以归因到有氧运动不够。
    * 图片阐述了人群健康程度和风险的分级
    * 久坐少动状态—死亡风险和疾病风险越高
    * 长期运动状态一死亡风险和疾病风险越低
  * 讨论：定期进行身体活动有哪些健康益处？
    * 儿童和青少年
    * 成年人和老年人
  * 运动的好处
    * 儿童和青少年
      * 运动对人类的益处很多，有非常多的相关研究，本次指南首次更新3—6岁的更新，改善骨骼健康和体重状态。
      * 疫情后复课后出现有些小孩运动中骨折的情况，骨密度过低，典型的运动缺乏。
      * 肥胖的小孩也居多
      * 6—17岁，对小孩的认知能力也会有提升，提高学习成绩。
      * 小孩上了小学之后，运动缺少，吃的增多，6—17岁小孩会迅速长胖（奶奶觉得你饿）
      * 造成脂肪细胞快速分裂，6—17岁时期控制好体重，会影响以后的身形状态和身形控制。
      * 缺乏沟通，遇到挫折和问题，无法自我调节，容易产生抑郁症状。通过运动可以减轻抑郁症状。
