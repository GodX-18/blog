# 文档加密功能使用指南

## 功能简介

已成功实现文档加密功能，可以为特定页面设置访问密码。

## 核心功能

✅ **配置文件加密设置** - 在 config.mts 中配置加密规则  
✅ **密码验证弹框** - 简洁的密码输入界面  
✅ **本地存储记忆** - 验证后自动记住状态  
✅ **路径匹配支持** - 精确匹配和通配符匹配  

## 测试方式

1. 访问测试页面：`/test-encrypt.html`
2. 输入密码：`test123`
3. 验证加密功能是否正常

## 技术架构

- **SimpleEncrypt.vue** - 简化的加密验证组件
- **MyLayout.vue** - 集成加密功能的布局组件  
- **配置驱动** - 通过 config.mts 统一管理加密规则
